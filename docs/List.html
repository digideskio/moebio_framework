Class: List

List

List is an Array with a type property. Lists have a number of methods to assist with working with them. There are also a number of helper functions in associated namespaces.

Additional functions that work on List can be found in:

  • Operators: ListOperators
  • Conversions: ListConversions
  • Generators: ListGenerators
