(static) bezier(x0, y0, cx0, cy0, cx1, cy1, x1, y1)

Parameters

Name Type Description
x0 Number

Starting x position.
y0 Number

Starting y position.
cx0 Number

First curve control point x position.
cy0 Number

First cure control point y position.
cx1 Number

Second curve control point x position.
cy1 Number

Second cure control point y position.
x1 Number

Ending x position.
y1 Number

Ending y position.

Example

bezier(10, 10, 10, 0, 40, 0, 40, 10);

(static) bezierM(x0, y0, cx0, cy0, cx1, cy1, x1, y1, d) → {Boolean}

Draws a mouse-enabled bezier curve using bezierCurveTo.

Parameters

Name Type Description
x0 Number

Starting x position.
y0 Number

Starting y position.
cx0 Number

First curve control point x position.
cy0 Number

First cure control point y position.
cx1 Number

Second curve control point x position.
cy1 Number

Second cure control point y position.
x1 Number

Ending x position.
y1 Number

Ending y position.
d Number

Distance away from line to count towards mouse interaction.

Returns Boolean

Returns true if the mouse is over the line on the current iteration of the cycle function.

Example

setStroke('black');

 setStroke('red');
 bezierM(10, 10, 10, 0, 40, 0, 40, 10);
}
