Parameters

Name Type Description

containing the aggregation list and lists to be aggregated

index of the aggregation list on the table

indexs of the lists to be aggregated; typically it also contains the index of the aggregation list at the beginning, to be aggregated using mode 0 (first element) thus resulting as the list of non repeated elements

9:most common element
10:random element
11:indexes
12:count non repeated elements
13:enlist non repeated elements
14:concat elements (string)
15:concat non-repeated elements
newListsNames StringList

optional names for generated lists

(static) buildDecisionTree(variablesTable, supervised, supervisedValue, min_entropy, min_size_node, min_info_gain, generatePattern, colorScale) → {Tree}

How to use (click to copy): mo.TableOperators.buildDecisionTree( variablesTable, supervised, supervisedValue, min_entropy, min_size_node, min_info_gain, generatePattern, colorScale )

builds a decision tree based on a table made of categorical lists, a list (the values of a supervised variable), and a value from the supervised variable. The result is a tree that contains on its leaves different populations obtained by iterative filterings by category values, and that contain extremes probabilities for having or not the valu ein the supervised variable. [!] this method only works with categorical lists (in case you have lists with numbers, find a way to simplify by ranges or powers)

Parameters

Name Type Description
variablesTable Table

predictors table
supervised Object

variable: list, or index (number) in the table (in which case the list will be removed from the predictors table)
supervisedValue Object

main value in supervised list (associated with blue)
min_entropy Number

minimum value of entropy on nodes (0.2 default)
min_size_node Number

minimum population size associated with node (10 default)
min_info_gain Number

minimum information gain by splitting by best feature (0.002 default)
generatePattern Boolean

generates a pattern of points picturing proprtion of followed class in node
colorScale ColorScale

to assign color associated to probability (default: blueToRed)

Returns Tree

tree with aditional information on its nodes, including: probablity, color, entropy, weight, lift tags:ds
