(static) addPageRankToNodes(network, From, useRelationsWeigh)

How to use (click to copy): mo.NetworkOperators.addPageRankToNodes( network, from, useRelationsWeight )

Adds PageRank as fromPageRank and toPageRank properties See Page Rank for more details. fromPageRank or toPageRank will be added as propertie to Nodes. I use two different pageranks, since a Network whose relations measure influence would require a pagerank to measure nodes influence into the system.

Parameters

Name Type Default Description
network Network
From Boolean true

Optional, default:true, to set if the PageRank uses the in-relations or out-relations
useRelationsWeigh Boolean false

Optional, default:false, set to true if relations weight will affect the metric balance, particularly interesting if some weights are negative tags:analytics,transformative

(static) buildNetworkClusters(network, mode, dendrogramTree, minWeight, addColorToNodes) → {Table}

How to use (click to copy): mo.NetworkOperators.buildNetworkClusters( network, mode, dendrogramTree, minWeight, addColorToNodes )

Builds a Table of clusters, based on an dendrogram Tree (if not provided it will be calculated), and a weight bias

Parameters

Name Type Description
network Network
mode Number

cluster mode, 0: Louvain (default), 1: Moebio algorithm
dendrogramTree Tree

Dendrogram Tree, if precalculated, changes in weight bias will perform faster (not required for Louvain)
minWeight Number

Weight bias, criteria to group clusters (0.5 default, the higher the value, the higer the number of clusters, not required for Louvain)
addColorToNodes Boolean

if true, adds a color associated to cluster, [!] modifies nodes color property

Returns Table

List of NodeLists tags:analysis
